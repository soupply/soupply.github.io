# Set commands enabled

Encode/decode this packet in [Sandbox](../../../sandbox/pocket102#Play.SetCommandsEnabled)

**Id**: 59

**Id** (hex): 3B

**Id** (bin): 00111011

**Clientbound**: ✔️

**Serverbound**: ✖️

Indicates whether the cheats are enabled. If not the client cannot send commands.
