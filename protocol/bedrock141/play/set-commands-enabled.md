# Set commands enabled

Encode/decode this packet in [Sandbox](../../../sandbox/bedrock141#play.set_commands_enabled)

**Id**: 59

**Id** (hex): 3B

**Id** (bin): 00111011

**Clientbound**: ✔️

**Serverbound**: ✖️

Indicates whether the cheats are enabled. If not the client cannot send commands.

Name | Type

enabled | bool
